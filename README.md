# 🧠 Brain Tumor Detection using CNN

A deep learning project that detects brain tumors from MRI images 
using Convolutional Neural Networks (CNN) for texture extraction.

## 📌 Project Overview
- Detects **Glioma Tumor** vs **No Tumor** from MRI scans
- Uses **CNN** for automatic texture feature extraction
- Achieves high confidence predictions
- Based on Texture-Aware MRI Brain Image Analysis

## 🛠️ Libraries Used
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

## 🚀 How to Run
1. Install libraries: `pip install tensorflow opencv-python matplotlib numpy scikit-learn`
2. Open `brain_tumor_detection.ipynb` in Jupyter / VS Code
3. Run all cells in order

## 📊 Results
- ✅ High accuracy tumor detection
- ✅ Confidence % shown for each prediction
- ✅ Confusion matrix and accuracy/loss graphs included
